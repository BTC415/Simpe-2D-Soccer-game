
# Ballzone - 2D Soccer Game

Real-time game with 2 teams where players try to score a goal and win (like in soccer or hockey).
## Created using

- Next.JS
- WebRTC
- Socket.io
- TailwindCSS
- Framer-motion
## Demo

https://ballzone.herokuapp.com
## Installation

Install like others Next.js apps using yarn
